## ヒヤリ・事故報告書管理システム

ヒヤリハット・事故報告書とは、介護業界で使用されている報告書です。ヒヤリハットとは、 例えば介護をしていて「ヒヤッとする場面」です。事故は「起きてしまった事故」の事を指 します。介護業界では、このヒヤリハット・事故報告書を積極的に活用する事で、介護事故 のリスクマネジメントを行い、可能な限り大きな事故を未然に防ぐ工夫をしています。
